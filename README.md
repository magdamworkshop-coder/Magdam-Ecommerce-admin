# MagDam Ecommerce Admin

Development admin dashboard for MagDam Ecommerce.

## Setup

Install dependencies:

```bash
npm install
```

## Run (development)

Start the Vite dev server:

```bash
npm run dev
```

Open http://localhost:5175/ in your browser.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Notes

- Uses Vue 3, Vite, and Tailwind CSS.
- If you need the dev server accessible on the network, run `vite --host` or set `--host` in `npm run dev`.
